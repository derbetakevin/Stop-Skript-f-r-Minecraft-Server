README

------------
INSTALLATION
------------

1. Installiere dir das Skript Plugin auf deinen Server
    https://dev.bukkit.org/projects/skript

2. Starte deinen Server einmal komplett neu

3. Füge die .sk Dateien aus der ZIP in den Ordner /MinecraftServer/plugins/Skript/scripts
WICHTIG: Beide Dateien müssen installiert werden, es reicht nicht, wenn nur eine Datei auf dem Server vorhanden ist!!!

4. Reloade alle Skripts mit /sk reload all

Damit ist die Installation abgeschlossen

------------
BEFEHLE
------------

/stop = Lässt den Spieler Kicken und es erscheint eine Nachricht auf dem Server, dass der Spieler gekickt wurde
/consolestop oder /cstop oder /cs = Stoppt den Server wirklich
